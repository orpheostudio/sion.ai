Sobre nós

Inteligência feita no Brasil,
para o Brasil.

A Telos Technologies nasceu de uma convicção simples: o Brasil não precisa depender de modelos estrangeiros para ter acesso a inteligência artificial de qualidade.

Somos uma startup brasileira de IA com sede no Brasil, construindo modelos, infraestrutura e produtos voltados para a realidade das empresas e pessoas que falam português.

Nossa missão

Tornar a IA acessível
para quem fala português

"Tecnologia de ponta que fala a língua do povo brasileiro — não uma tradução, mas uma criação original para o nosso contexto, cultura e legislação."

Quando você usa um modelo de IA genérico em português, está usando uma ferramenta construída para o inglês e adaptada às pressas. O Claudinei foi construído de dentro para fora: corpus brasileiro, infraestrutura nacional, conformidade com a LGPD desde o primeiro dia e uma equipe que entende o que significa fazer negócios no Brasil.

Nossa história

Uma trajetória de construção

Início
AmplaAI Platforms
Primeiros passos: construção do site Claudinei sob a marca AmplaAI, com estrutura de 14 arquivos, conformidade LGPD, ads.txt para AdSense e domínio claudinei.ia.br.
Motor
Java AI Backend — Transformer Decoder
Desenvolvimento de um sistema GPT-style em Java com ~25M parâmetros, backpropagation real, AdamW optimizer, JWT auth, Redis rate limiting e Docker com Traefik HTTPS. API compatível com OpenAI.
Produtos
Família Claudinei & S.ai+ Apps
Expansão para múltiplos produtos: Sion AI (PWA com Mistral), Tinta (rede literária social), Aion AI (chat estilo ChatGPT), app Android para Google Play com Supabase e AdMob.
Hoje
Telos Technologies — Motor Sena V1.0
Renomeação da empresa para Telos Technologies (Tellos Tecnologia LTDA.), identidade visual completa em azul-marinho e ouro, lançamento do Motor Sena V1.0 e abertura da API para desenvolvedores.
O que nos guia

Quatro princípios inegociáveis

01
Brasilidade primeiro
Cada decisão de produto começa com a pergunta: isso faz sentido para o brasileiro? Não para o usuário genérico de IA, mas para quem vive, trabalha e pensa em português.
02
Engenharia séria
Não usamos wrappers de API de terceiros e chamamos de IA. Construímos o modelo, a infraestrutura e os produtos. Do Transformer Decoder ao app no Play Store — tudo de dentro para fora.
03
Privacidade por design
LGPD não é um checkbox para nós. É uma restrição de arquitetura. Dados processados no Brasil, sem compartilhamento para treino, com controle total do usuário.
04
Acesso democrático
IA de qualidade não deve ser privilégio de grandes corporações. O plano gratuito existe porque acreditamos que qualquer desenvolvedor brasileiro merece experimentar o estado da arte.
